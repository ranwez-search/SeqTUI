# Changelog

## 0.1.1 - 2026-01-14
- NEXUS partition format (IQ-TREE compatible) for the optional partition file generated during supermatrix construction.

## 0.1.0 - 2025-12-29
Initial public release.

### Features
- Interactive terminal viewer for nucleotide and protein sequences
- Support for aligned and unaligned sequences
- On-the-fly translation and FASTA export
- Export isolated biallelic SNPs to VCF
- Vim-style navigation and interactive search
- Command-line mode for batch convert/translate/combine (pipe-friendly)
- Designed for large datasets / HPC usage
